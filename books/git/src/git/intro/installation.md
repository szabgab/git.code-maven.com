# Git Installation

On **Linux** you use your package manger (apt-get, yum, etc...) or install from [git-scm](https://git-scm.com/)

* `yum install git-core`
* `apt-get install git-core`

On **Microsoft Windows** install Git from [git-scm](https://git-scm.com/) or [Git for Windows](https://gitforwindows.org/). They are the same.

On **MacOS** use [Homebrew](https://brew.sh/) or [git-scm](https://git-scm.com/).

* `brew install git`


* In some systems you could also just type `git` in the Terminal and you'll be offered to install `git`.
